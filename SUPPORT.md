# Support

The Health Game is an open source project in MVP development.

## Getting Help

Use the repository's issue tracker for:

- Reproducible bugs.
- Documentation problems.
- Small, scoped enhancement proposals.
- Questions about local setup that are not already answered in the docs.

Before opening an issue:

- Search existing issues and pull requests.
- Check [README.md](README.md).
- Check [CONTRIBUTING.md](CONTRIBUTING.md).
- Include the backend or frontend version details relevant to the problem.

## Security Issues

Do not request security support through public issues. Follow
[SECURITY.md](SECURITY.md).

## Maintainer Availability

Maintainers are not expected to provide private consulting, guaranteed response
times, or support for unrelated environments. Clear reproduction steps and small
examples make community support more likely.
